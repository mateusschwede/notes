Converter Csv em Json com Php

Código Php de conversão de arquivos Csv em Json

Pré-requisitos

Necessário ter o Xampp executando, criar diretório no htdocs para armazenar os arquivos de código e csv. Após, baixe o arquivo .csv, nomeando-o planilha.csv no mesmo diretório do .php. Caso pretendes modificar o nome do .csv, atualize-o também no .php! Precisa de inspiração pra baixar arquivos .csv? O governo federal dispõe dados públicos, nesse formato. Acesse abaixo.

Dados públicos >

codigo.php


<?php
function csvToJson($file) {
    if (!($fp = fopen($file,'r'))) {die("Não abriu");}
    $key = fgetcsv($fp,"1024",",");    
    $json = array();
    while ($row = fgetcsv($fp,"1024",",")) {$json[] = array_combine($key,$row);}
    fclose($fp);
    return json_encode($json);
}
header('Content-Type: application/json');
print(csvToJson("planilha.csv"));
