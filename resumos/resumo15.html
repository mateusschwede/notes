Bootstrap

Biblioteca Frontend CDN

O Bootstrap é uma biblioteca frontend de componentes UI previamente prontos, onde pode ser utilizada em conjunto de soluções Backend, complementando o Frontend, a fim da menor utilização e gasto de tempo com estilização de conteúdos. Fornece uma gama de componentes muito variada e tecnológica, favorecendo o desenvolvimento de aplicações responsivas, de forma ágil e com pouco código. A maioria dos componentes Bootstrap econtram-se no atributo class, sendo nesse ponto a localização de suas propriedades.

Instalação

Para executar o Bootstrap em seu sistema, de forma CDN (Simplesmente import), basta importar, no <header> de seu Html, os códigos abaixo. 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-eOJMYsd53ii+scO/bJGFsiCZc+5NDVN2yr8+0RDqr0Ql0h+rP48ckxlpbzKgwra6" crossorigin="anonymous">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/js/bootstrap.bundle.min.js" integrity="sha384-JEW9xMcG8R+pH31jmWH6WWP0WintQrMb4s7ZOdauHnUtxwoG2vI5DkLtS3qm9Ekf" crossorigin="anonymous"></script>
<meta name="viewport" content="width=device-width, initial-scale=1">

Layout

O Booststrap trabalha, principalmente, com Grid Layout, onde a tela é dividida em linhas <row>, sendo cada linha dividida em até 12 colunas <col>. Cada coluna pode possuir, em seu interior, linhas e colunas, respectivamente. O exemplo a seguir dividirá a tela em 2 linhas, onde a 1ª possui 4 colunas e a 2ª 3 colunas. 
    
<div class="container-fluid">
    <div class="row">
        <div class="col-sm-3">
            Linha 1, coluna 1
        </div>
        <div class="col-sm-3">
            Linha 1, coluna 2
        </div>
        <div class="col-sm-3">
            Linha 1, coluna 3
        </div>
        <div class="col-sm-3">
            Linha 1, coluna 4
        </div>
    </div>
    <div class="row">
        <div class="col-sm-3">
            Linha 2, coluna 1
        </div>
        <div class="col-sm-3">
            Linha 2, coluna 2
        </div>
        <div class="col-sm-3">
            Linha 2, coluna 3
        </div>
    </div>
</div>

Breadcrumb

Menus Breadcrumb (Migalhas de Pão) representam o caminho de páginas em que o usuário está situado. 

<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a href="#">Pagina1</a></li>
        <li class="breadcrumb-item"><a href="#">Pagina2</a></li>
        <li class="breadcrumb-item active" aria-current="page">PaginaAtual</li>
    </ol>
</nav>
